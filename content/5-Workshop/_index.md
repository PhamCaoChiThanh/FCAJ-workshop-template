### Workshop: End-to-End Serverless Deployment of SmartDorm on AWS

Welcome to the hands-on workshop lab for the **SmartDorm** system.

In this technical lab, you will design the PostgreSQL relational database schema, write Terraform Infrastructure as Code (IaC) templates, develop a C# ASP.NET Core Web API running serverless on AWS Lambda, integrate Amazon S3 for media uploads, and secure Next.js administrative pages using client-side URL Masking hosted on Vercel.
