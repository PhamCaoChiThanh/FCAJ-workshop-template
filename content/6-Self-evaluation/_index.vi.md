## Tự đánh giá quá trình thực tập

Dưới đây là bảng tự đánh giá kết quả rèn luyện của bản thân qua 12 tuần thực tập tại AWS Vietnam dưới vai trò FCJ Cloud Intern:

| Tiêu chí | Đánh giá | Nhận xét chi tiết |
| :--- | :---: | :--- |
| **Kiến thức chuyên môn** | **Tốt** | Làm chủ được việc thiết kế kiến trúc Serverless, sử dụng Terraform IaC và viết Web API bằng C# .NET. |
| **Khả năng tự học** | **Tốt** | Tự tìm hiểu và tích hợp thành công SDK S3 để sửa lỗi hệ thống file read-only của Lambda. |
| **Tính chủ động** | **Tốt** | Chủ động đưa ra giải pháp loại bỏ NAT Gateway giúp giảm chi phí hạ tầng về gần 0 USD. |
| **Kỷ luật & Tác phong** | **Tốt** | Lên văn phòng đầy đủ số buổi quy định, hoàn thành nhiệm vụ đúng tiến độ cam kết. |
| **Kỹ năng giao tiếp** | **Khá** | Giao tiếp rõ ràng với nhóm dự án, giải thích tốt các giải pháp kiến trúc đám mây. |
| **Làm việc nhóm (Teamwork)**| **Tốt** | Phối hợp ăn ý trong việc kết nối giao diện Next.js với backend API. |
| **Giải quyết vấn đề** | **Tốt** | Giải quyết triệt độ lỗi CORS phức tạp giữa Vercel client và Lambda endpoint. |
| **Đóng góp cho dự án** | **Tốt** | Xây dựng thành công hệ thống quản lý KTX SmartDorm hoàn chỉnh chạy thực tế trên Cloud. |
