### Các bài viết kỹ thuật chia sẻ kiến thức đã đăng trên AWS Study Group

Trong quá trình thực tập, nhóm đã nghiên cứu sâu và chia sẻ các giải pháp hữu ích lên cộng đồng học tập AWS:

*   [Bài viết 1: Tối ưu chi phí hạ tầng SmartDorm về gần 0 USD](3.1-Blog1/)
    *   *Nội dung:* Phân tích cách tối ưu hóa kiến trúc AWS (Lambda Serverless, HTTP API Gateway, RDS Free Tier, và loại bỏ NAT Gateway) để giảm thiểu hóa đơn đám mây về mức tối thiểu.
*   [Bài viết 2: Giải quyết lỗi hệ thống file chỉ đọc (Read-only) của AWS Lambda với Amazon S3](3.2-Blog2/)
    *   *Nội dung:* Bài viết chia sẻ kinh nghiệm xử lý lỗi ghi đĩa khi chuyển đổi backend C# truyền thống lên chạy không máy chủ (Serverless), hướng dẫn tích hợp Amazon S3 SDK.
*   [Bài viết 3: Triển khai Hạ tầng dưới dạng mã (IaC) đơn giản bằng Terraform trên AWS](3.3-Blog3/)
    *   *Nội dung:* Hướng dẫn từng bước viết code Terraform để tự động hóa việc khởi tạo VPC, Subnet, Security Groups, Lambda và RDS PostgreSQL.
